- Built an offline-to-online RL pipeline for AGV navigation using Nav2 critic-based (A* + DWB) expert data and TD3-BC initialization with domain randomization, outperforming training from scratch in controlled evaluations.
- Developed an LLM-assisted Hierarchical Potential-based Reward Shaping (HPRS) framework for automated failure diagnosis and reward refinement, enabling systematic policy improvement.
